# Bitcoin E-Commerce - CUBO+

**Proyecto:** Carrito de compras con facturación en Bitcoin para el programa CUBO+.

## 🛒 Funcionalidades
- Cargar productos desde un archivo CSV.
- Agregar productos a un carrito de compras.
- Generar facturas en PDF con totales en BTC.
- Simular pagos con Bitcoin.
